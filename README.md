# Supply Chain & Inventory Analysis Dashboard

This project analyzes supply chain operations to identify:

- Top selling products
- Supplier performance
- Shipping efficiency
- Inventory risks
- Revenue trends

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit

## Features
- Interactive filters
- KPI metrics
- Revenue analysis
- Supplier lead time analysis
- Stock risk detection

## Dashboard

Built using Streamlit for interactive analytics.
